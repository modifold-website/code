## Endpoints

Version 2 endpoints use the `/v2` prefix and can be used independently from the project-scoped v1 routes.

| Method | Path | Summary |
| --- | --- | --- |
| `GET` | [`/v2/version/{versionId}`](/api-v2/get-version-versionid) | Get a version directly by its ID |
